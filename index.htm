How to use the treemap


Levels

There are two treemaps, one for Java tests and the other for C++ tests. The treemaps are broken down to three levels.

  • Level 1: Vob
  • Level 2: Package
  • Level 3: TestCase

To drill down to the next level, simply click on the box. To drill up, use the button on the left available when you are not at the bottom level.

Colors

Each rectangle in Level 3 is filled with either green or red.

  • Green = Test passed
  • Red = Test failed

When on Levels 1 or 2, the color scale for the rectangle will be the average fail score for the package or vob ranging from 0 - 100, where 100 is a completely failing vob or package.

How to use the parallel coordinates plot


Each parallel axis on the plot shows different information about a particular testcase for a given day.


Filtering tips:

  • Data points can be filtered by a click and drag action along any axis.
  • Each axis can have its own filter.
  • Filters can be removed by clicking anywhere outside the filter box on the axis.

Plot and table tips:
  • Reordering axes can be done by dragging an axis title to a new location.
  • Hovering over any row on the table will highlight the data point in the parallel coordinates plot.
  • Sorting a particular column on the table will change the rendering order on the plot.
  • Any clicks along an axis will cause the plot to render.
